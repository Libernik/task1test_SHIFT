Утилита фильтрации содержимого файлов

Версия JAVA - 22 SDK
Система сборки - Maven 3.9.8
Плагин для сборки jar-ов от Maven - https://mvnrepository.com/artifact/org.apache.maven.plugins/maven-jar-plugin

Инструкция по запуску:
Утилиту можно запустить с помощью команды java -jar .\task1test-1.0-SNAPSHOT.jar -s -a -p sample- in1.txt in2.txt
Либо с помощью команды java "полный путь к файлу Main.java" -s -a -p sample- in1.txt in2.txt

Аргументы:
-a Режим добавления в существующие файлы. Если файла не существует, то он будет создан
-s Краткая статистика по считанным данным (выбрана по умолчанию)
-f Полная статистика. Необходимо выбрать только 1 режим статистики!
-p Префикс имён выходных файлов. После него необходимо указать имя, которое будет добавлено к каждому файлу вывода.
-o Путь вывода файлов. После него необходимо указать папку, где будут созданы файлы вывода.
