Утилита фильтрации содержимого файлов

Версия JAVA - 22 SDK
Система сборки - Maven 3.9.8

Инструкция по запуску:

Аргументы:
-a Режим добавления в существующие файлы. Если файла не существует, то он будет создан
-s Краткая статистика по считанным данным (выбрана по умолчанию)
-f Полная статистика. Необходимо выбрать только 1 режим статистики!
-p Префикс имён выходных файлов
-o Путь вывода файлов (папка, где будут созданы файлы)
